# Embedded Assets

`NotoSansSCSubset.ttf` is embedded to provide a more modern Chinese UI font for
AirPlay music metadata rendered through LVGL TinyTTF.

Source:
- Base font: https://github.com/google/fonts/raw/main/ofl/notosanssc/NotoSansSC%5Bwght%5D.ttf
- Subset generated with HarfBuzz `hb-subset`.

License:
Noto Sans SC is distributed under the SIL Open Font License 1.1.

Subset coverage:
Basic Latin, common punctuation and symbols, Hiragana/Katakana, fullwidth forms,
GB2312 Chinese characters, plus U+6000-U+9FFF for broader music metadata coverage.
